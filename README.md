Project Overview

This project analyzes customer churn data to identify the key factors influencing customer retention and churn. The analysis combines SQL, Python, and data visualization techniques to uncover business insights that can help organizations improve customer retention strategies.

The project demonstrates the complete data analysis workflow, including database connectivity, data cleaning, exploratory data analysis (EDA), feature engineering, KPI generation, and visualization.

Objectives
Analyze customer churn behavior.
Identify high-risk customers.
Understand factors contributing to churn.
Generate business insights using visualizations.
Create customer risk categories for better decision-making.
